# Task 10: Notifikasi In-App (Nice to Have)

**Deskripsi:**
Sistem notifikasi dalam aplikasi untuk memberi tahu pengguna mengenai perubahan status.

**Spesifikasi:**
- Buat migration dan model untuk tabel 
otifications.
- Tabel 
otifications: id, user_id, title, message, is_read, timestamps.
- Buat helper/service untuk memancarkan notifikasi (contoh: notifikasi ke Admin saat HR ajukan lowongan, notifikasi ke Pelamar saat status lamaran berubah).
- Buat widget lonceng notifikasi di UI template.
- Sediakan data dummy (Seeder).
