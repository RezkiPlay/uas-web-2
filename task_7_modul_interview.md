# Task 7: Modul Penjadwalan Interview (Nice to Have)

**Deskripsi:**
Fitur lanjutan untuk mengatur jadwal wawancara kandidat.

**Spesifikasi:**
- Buat migration dan model untuk tabel interview_schedules.
- Tabel interview_schedules: id, application_id, schedule_time, location_or_link, notes, timestamps.
- HR dapat memindahkan status lamaran ke interview sambil membuat form penjadwalan.
- Pelamar dapat melihat jadwal wawancara di dashboard-nya.
- Sediakan data dummy (Seeder) untuk jadwal wawancara.
